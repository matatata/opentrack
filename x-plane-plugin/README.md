# X-Plane plugin

Copy the whole opentrack directory into X-Plane's plugin directory at `<X-Plane-Root>/Resources/plugins/`. Please remove any pre-existing plugin files (named opentrack.xpl or similar) from that plugin directory.

In opentrack you'll need to use the Output protocol *opentrack 2.0 Enhanced*. The previos *Wine/X-Plane* is replaced by that. You migh need to adjust for a reversed X-Axis.

## macOS Notes

On macOS this is now a universal binary for Intel and Apple Silicon.
